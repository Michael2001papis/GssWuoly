# GssWuoly – Game Hub

פלטפורמת משחקים מבוססת דפדפן. איקס עיגול, נחש, מערכת התחברות, טבלאות מובילים והעדפות נגישות.

---

## טכנולוגיות ותשתית

- **HTML5, CSS3, JavaScript** – ללא פריימוורק, קוד וונילי
- **Vite** – בנייה ופיתוח (הרצה מקומית, build לפרודקשן)
- **Google Fonts (Heebo)** – גופן עברי
- **LocalStorage** – שמירת משתמש, ניקוד, העדפות
- **EmailJS** – שליחת מייל אימות (אופציונלי)

---

## מבנה הפרויקט

```
תיכנונים/
├── index.html              # הפניה לדף הבית
├── package.json
├── vite.config.js
├── sitemap.xml
├── assets/
│   └── favicon.svg
├── css/
│   └── app.css             # עיצוב משותף (topbar, modal, theme)
├── js/
│   ├── app-common.js       # תפריט עוד, שעון, תאורה, נגישות, session
│   ├── auth.js             # התחברות/הרשמה (LocalStorage)
│   ├── sounds.js           # אפקטי שמע
│   └── toast.js            # הודעות
└── pages/
    ├── home/               # דף בחירת משחק
    ├── about/              # אודות
    ├── contact/            # צור קשר
    ├── privacy/            # מדיניות פרטיות
    ├── terms/              # תנאי שימוש
    ├── 404/                # דף לא נמצא
    ├── login/               # התחברות והרשמה (+ email-config)
    └── games/
        ├── tic-tac-toe/     # איקס עיגול – PvP / PvC
        └── snake/          # נחש – טיימר, ניקוד, מובילים
```

---

## תפקידי הקבצים המרכזיים

| קובץ | תפקיד |
|------|--------|
| `app-common.js` | תפריט "עוד", שעון, מצב תאורה, הגדרות נגישות, מעקב פעילות, timeout |
| `auth.js` | רישום, התחברות, התנתקות (LocalStorage) |
| `sounds.js` | צלילי משחק – הפעלת השמעה רק אחרי אינטראקציה |
| `toast.js` | הודעות קצרות במקום `alert` |
| `email-config.js` | קונפיגורציה ל-EmailJS (בתיקיית login) |

## פקודות הפעלה

```bash
# התקנת תלויות
npm install

# הפעלת שרת פיתוח (localhost)
npm run dev

# בנייה לפרודקשן
npm run build

# הצגת הבנייה (לאחר build)
npm run start

# או
npm run preview
```

## דרישות

- **Node.js** (מומלץ 18+)
- **npm**

## איך להריץ

1. פתח טרמינל בתיקיית הפרויקט
2. הריץ `npm install` (פעם אחת)
3. הריץ `npm run dev`
4. פתח בדפדפן את הכתובת שמופיעה (בדרך כלל `http://localhost:5173`)
